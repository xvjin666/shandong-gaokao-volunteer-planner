# 贡献指南

感谢你愿意改进这个项目。这个仓库的定位是本地优先、可解释、可复核的山东高考志愿决策引擎。

## 开发环境

```bash
python -m pip install -e ".[test]"
python -m pytest
```

如需处理 Excel：

```bash
python -m pip install -e ".[excel]"
```

## 提交前检查

- 运行 `python -m pytest`。
- 不提交真实数据库、官方网页镜像、客户资料、日志、备份和交付包。
- 新增数据样例必须是合成数据，或已经确认具备可公开发布授权。
- 修改推荐、位次换算、数据导入和筛选逻辑时，请补充或更新测试。

## 数据贡献原则

请不要在公开 issue 或 pull request 中上传完整官方文件、第三方网页镜像、客户数据、真实填报记录或任何未经授权的数据集。

如果你想贡献数据处理能力，请优先贡献：

- 字段映射规则
- 数据校验逻辑
- 下载/解析脚本
- 小型合成样例
- 来源清单和复核说明

## 责任边界

本项目输出仅供辅助分析。正式填报前必须以山东省教育招生考试院、高校官方招生计划、招生章程和人工复核为准。
